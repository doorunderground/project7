# RC카 자율주행 프로젝트

## 프로젝트 개요
라즈베리파이 기반 RC카가 트랙 위의 차선(노란 외곽선 + 흰색 중앙 점선)을 인식하고
자율주행하는 시스템을 구현한다.

트랙 특징:
- 검정 아스팔트 위에 노란 실선(외곽)과 흰색 점선(중앙)으로 구성
- S자 커브, 직선 구간 혼합, RC카가 겨우 지나갈 정도로 좁음


## 구현 방향

1.  차선을 따라서 자율주행 기능 (양방향)
1.1 불안정한 차선 유지를 차선(lane) detect 기능을 추가해서 안정성을 유지하도록 함

2.  객체(장애물)이 인식되면 회피 주행

3.  횡단보고를 인식하면 일시 정지 후 주행



### 차선만으로 자율주행이 어려운 이유
- 세그멘테이션 마스크만으로는 "핸들을 얼마나 꺾어야 하는지" 제어값 계산이 불안정
- 특히 S자 커브처럼 곡률이 큰 구간에서 오류 가능성 높음
- **결론: 모방학습(STEP 1) + 차선인식(STEP 2)을 함께 써야 안정적**
-> 모방학습이란? : 보상 설계에 따라 스스로 탐색하는 강화학습과 달리
                  시연 데이터(정답 행동)을 그대로 모사


## 구현1 

### STEP 1 - 모방 학습 데이터 수집
- 파일:   `Pc_server(wasd).py` + `Pi_drive(wasd).py`   WASD 조종하면서 데이터 수집
- 학습:   `1train.wasd.py`
- 테스트: `test_auto_drive(wasd).py`

- UDP 통신 (라즈베리파이 -> PC: 영상,   PC- > 라즈베리파이: 모터 명령)
- 저장 경로:
  -  1차 (한 방향): `collected_data/`
  -  2차 (양 방향): `collected_data2/`
- 저장 형식:  `frame_000001.jpg` + `frame_000001.txt`
  -  txt 1줄 : 모터값 (예: `-80,-20`)
  -  txt 2줄: 키 조합 이름 (예: `wd`)
- **stop 프레임은 저장 안 함** → 클래스 불균형 방지
